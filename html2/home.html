Scraper Web

This is an application "web scraper" that aims to scrap the web. The application then provides the ability to conduct research on the Web pages found by the "web scraper". It also provides a number of interesting statistics on the web page found.

Home

The main page of the application. You are currently reading this page. This page serves no purpose except to present the application and its various functionalities. Here is a presentation of each application tabs.

Indexing

The application will index the web iteratively from the given url. It is possible to give a maximal number of url to extract links. If the number of links is specified by "*", the application will look continuously until it finds no links.

Search

This page allows you to perform searches on the indexed pages. It can be searched by two criteria. The first search the word in the page body, the second search the word in url. The searching concerne only scraped pages.

Statistic

This page gives some statistics on the pages that have been indexed and the research done.

List

This page lists the known application url. It shows, in green, pages whose links has already been extracted and ,in red, pages whose the links extraction has failed.

Code

This application is coded in html / javascript client side and it use nodejs server side. You can find the source code of the application on github. Do not hesitate to post all your improvement ideas.
